## zh-CN

## en-US

File Upload.

````jsx
import { Upload } from 'choerodon-ui/pro';

const props = {
  headers: {
    'Access-Control-Allow-Origin': '*',
  },
  action: '//localhost:3000/upload',
  multiple: true,
  accept: ['.deb', '.txt', '.pdf', 'image/*'],
  uploadImmediately: true,
};

ReactDOM.render(
  <div>
    <Upload {...props} />
  </div>,
  mountNode
);

````
