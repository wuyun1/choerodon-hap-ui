进度条。

## 何时使用

展示进度。

## API


Progress 的属性说明如下：

参数 | 说明 | 类型 | 默认值
-----|-----|-----|------
value | 当前百分比值，同percent,受控。 | number | -
dataSet | 数据源 | DataSet | -
name | 字段名 | string | -

更多属性请参考 [Progress](/components/progress/)。
