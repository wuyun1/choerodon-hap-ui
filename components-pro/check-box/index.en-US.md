多选框。

## 何时使用

- 在一组可选项中进行多项选择时；
- 单独使用可以表示两种状态之间的切换，和 `Switch` 类似。


## API

### Input\[type=checkbox\]

| 参数      | 说明                                     | 类型        |默认值 |
|-----------|------------------------------------------|------------|--------|
| unCheckedValue | 未选中时的值 | any  |  |
| unCheckedChildren | 非选中时的内容 | string\|ReactNode |  |
| indeterminate | 中间状态 | boolean  | false |

更多属性请参考 [Radio](/components-pro/radio/)。
