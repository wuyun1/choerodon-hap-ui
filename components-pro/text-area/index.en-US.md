## API


### TextArea

属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
| cols | 文本域宽 | number  | - |
| rows | 文本域高 | number  | - |
| resize | 是否能够拖拽调整大小，可选值： `none` `both` `vertical` `horizontal` | string  | none |

更多属性请参考 [FormField](/components-pro/field/#FormField)。
